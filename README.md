## bun
🚀 用bun启动速度快了好几倍
### 安装bun
```bash
npm -g install bun //相信我用了bun之后，这是你最后一次用npm😂
```



## Installation
```bash
$ bun install
```

## Running the app

```bash
# development
$ bun run start

# watch mode
$ bun run start:dev

# production mode
$ bun run start:prod
```
